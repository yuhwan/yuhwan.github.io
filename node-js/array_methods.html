              # 7. 유용한 배열 함수들

              지금까지 배운것처럼 자바스크립트에서는 함수를 다른 함수의 파라미터로 넘길 수 있습니다. 그렇기 때문에 콜백함수가 가능하고 이를 통해 자바스크립트는 비동기 방식의 함수들을 지원하고 있습니다. 하지만, 함수를 다른 함수의 파라미터로 넘길 수 있는 것의 장점은 여기에서 그치지 않습니다. 자바스크립트는 배열의 값을 다루는데 용이한 다양한 함수들을 제공하고 있습니다. 그리고 이 함수들 중에서 함수를 파라미터로 받는 함수들이 있습니다. 이 함수들중 가장 많이 사용하는 forEach, filter, map, reduce 네가지 함수를 배워보겠습니다. 이 함수들을 이용하면 보다 쉽고 깔끔하게 배열을 다룰 수 있습니다.

              # forEach

              시작하기 전에 배열의 값을 하나씩 콘솔에 출력하는 예제를 다시한번 떠올려 봅시다. for문을 이용해서 아래와 같이 코드를 작성할 수 있습니다.

              const arr = [1,2,3,4,5,6]
for(let i = 0; i < arr.length; i++) {
    console.log(`${i}: ${arr[i]}`)
}

              하지만 이 방식은 자바스크립트애서 배열을 다루는데 거의 사용하지 않는 방법입니다. 기본적으로 for문은 가독성이 많이 떨어지기 때문입니다. 훨씬 더 높은 가독성과 자유도를 제공하는 함수를 자바스크립트에서 제공합니다. 바로, forEach입니다. 먼저, forEach의 예제를 한번 살펴보도록 합시다.

              const arr = [1,2,3,4,5,6]
arr.forEach(function(item) {
    console.log(itme)
});

              forEach는 배열이 기본으로 제공하는 함수이기 때문에 배열의 변수명 뒤에 점(.)을 찍어서 호출하면 됩니다. forEach 함수는 파라미터로 함수를 하나 받습니다. forEach는 내부에서 배열의 아이템을 하나 하나씩 가져와서 첫번째 파라미터 (item)에 넣어 파라미터로 받은 함수를 불러줍니다. 간단히 forEach 로직만 구현해 보자면 아래와 같은 방식입니다.

              const arr = [1,2,3,4,5,6]
function forEach(fuc) {
    for(let i = 0; i < arr.length; i++) {
        func(arr[i])
    }
}

              forEach 함수는 반복문만 돌리는 것이기 때문에 리턴값이 존재하지 않습니다. 그리고 forEach 함수를 사용할때 화살표 함수 문법을 사용하면 훨씬 더 간결해지는 것을 확인하실 수 있습니다.

              const arr = [1,2,3,4,5,6]
arr.forEach(item => {
    console.log(item)
})

              forEach를 사용할 때, 짝수 번호의 인덱스의 값만 출력하고 싶다면 어떻게 해야할까요? forEach가 파라미터로 받는 함수는 첫번째 파라미터로 배열의 값을 받습니다. 여기서 그치지 않고, 두번째 파라미터로 해당 아이템의 인덱스도 같이 넘겨줍니다. 인덱스 값이 필요없다면 위와 같이 사용하면 되지만, 만약 필요하다면 아래와 같이 사용하면 됩니다.

              const arr = [1,2,3,4,5,6]
arr.forEach((item, index) => {
    console.log(`${index}: ${item}`)
})

              # filter

              두번째로 배워볼 배열의 함수는 filter 입니다. 이름 자체가 의미하듯이 필터는 배열에서 특정 조건에 해당하는 아이템만 담은 배열을 리턴해 주는 함수입니다. 이 filter 함수 역시 파라미터로 함수를 받습니다. 그리고 이 함수의 리턴값이 참이면 리턴되는 배열에 값을 포함시키고 거짓이면 빼는 방식으로 동작합니다. 3보다 큰값들만 배열에 담는 예제를 작성해 봅시다.

              const arr = [1,2,3,4,5,6]
const filteredArr = arr.filter(function(item) => {
    return item > 3
})
console.log(filteredArr)

              물론 화살표 함수를 사용하면 더 간단해 집니다.

              const arr = [1,2,3,4,5,6]
const filteredArr = arr.filter(item => {
    return item > 3
})

              기억하시겠지만, 화살표함수에서는 한줄로 리턴되는 값의 경우에는 중괄호도 생략할 수 있습니다. 결국 아래와 같이 굉장히 짧고도 가독성이 높은 코드를 작성할 수 있게 됩니다.

              const arr = [1,2,3,4,5,6]
const arrLargerThan3 = arr.fitler(_ => _ > 3)

              # map

              이번에 배울 함수는 map 입니다. map 함수 역시 이름이 말해주듯이 배열의 각 아이템을 특정 조건에 따라서 다른 값으로 매핑해주는 함수 입니다. 물론 매핑된 값들은 배열의 형태로 리턴됩니다. 예를 들어, 배열의 값을 모두 2배씩 크게 만들고 싶다면 다음과 같이 사용하면 됩니다.

              const arr = [1,2,3,4,5,6]
const arr2x  = arr.map(_ => _ * 2)
console.log(arr2x)
// [2,4,6,8,10,12]

              map 함수를 사용하는 예제가 잘 안떠오르실 수 있는데 저같은 경우에는 객체로 구성된 배열에서 각 객체에서 원하는 필드만 포함시키도록 변경하는데 map 함수를 자주 사용합니다. 예를 들어, 여러명의 사용자 정보를 담고 있는 배열이 있는데 저는 이 중에서 나이와 성별만 필요하다고 가정합시다. 이때 다음과 같이 코드를 작성하면 됩니다.

              const users = [
    { age: 20, name: 'Dan', gender: 'male', id: 123 },
    { age: 21, name: 'Chris', gender: 'male', id: 124 },
    { age: 23, name: 'David', gender: 'male', id: 125 }
]
const info = users.map(user => {
    const age = user.age

              # reduce

              마지막으로 배울 함수는 reduce 입니다. reduce는 지금까지 배운 함수들보다는 조금 복잡합니다. 의미를 이야기해보자면, 큰 배열을 원하는 하나의 값으로 수렴하게 되는 형태입니다. 예를 들어, 배열에 여러 숫자들이 있을때 합을 구하는 코드를 reduce 함수를 통해 구할 수 있습니다. 문법은 다른 함수들과 다르게 첫번째 파라미터에는 중간결과값이 들어가고 두번째 파라미터에는 순서에 해당하는 배열의 값이 들어값니다. 예제를 먼저 보도록 합시다.

              const arr = [1,2,3]
const result = arr.reduce(function(sum, item) {
    return sum + item
})
console.log(result) // 6

              어떻게 동작하는지 헷갈리실텐데 하나씩 살펴봅시다. 가장 기본적으로 동작하는 방식은 지금까지의 배열 함수들과 동일합니다. reduce 함수가 파라미터로 받은 함수는 배열을 돌면서 한번씩 불립니다. 문제는 sum 입니다. 배열의 아이템이 돌때 파라미터로 받은 함수에서 리턴된 (return sum + item)의 값이 다음번 함수의 파라미터 sum에 할당됩니다. 다만, 가장 처음에는 첫번째 아이템을 건너뛰고 두번째 아이템을 위한 콜을 할때 sum의 값을 첫번째 아이템으로 할당하게 됩니다. 조금 더 쉽게 이해하기 위해 콘솔 로직을 추가해서 결과값을 살펴봅시다.

              const arr = [1,2,3,4,5,6]
const result = arr.reduce(function(sum, item) {
    console.log(sum, item)
    return sum + item
})
/**
 * output:
 * 1, 2
 * 3, 3
 * 6, 4
 * 10, 5
 * 15, 6
 */

              첫번째 초기값을 직접 설정할 수 도 있습니다. 초기값은 reduce 함수의 두번째 파라미터로 전달해주면 됩니다. 예를 들어, 위와 똑같은 예제를 다음과 같이 작성할 수 있습니다.

              const arr = [1,2,3,4,5,6]
const result = arr.reduce(function(sum, item) {
    console.log(sum, item)
    return sum + item
}, 0)
/**
 * output:
 * 0, 1
 * 1, 2
 * 3, 3
 * 6, 4
 * 10, 5
 * 15, 6
 */

              이제 화살표 함수 문법을 이용해서 reduce 함수도 조금 깔끔하게 만들어 봅시다.

              const arr = [1,2,3]
const result = arr.reduce((sum, item) => sum + item)    
console.log(result)

              # 정리

              지금까지 배열의 유용한 함수들인 forEach, filter, map, reduce에 대해 살펴보았습니다. 이 함수들과 화살표 함수를 조합해서 사용하면 여러분의 코드가 훨씬 더 간결하면서도 높은 가독성을 유지할 수 있습니다. 지금부터는 배열의 값을 가져오기 위해서 for문의 사용을 지양해주세요. 위에서도 말했지만 for는 굉장히 가독성이 낮은 문법입니다. 이외에도 sort를 포함한 다양한 함수들이 존재합니다. 여러분이 처음부터 로직을 만들지 말고 있는 함수들을 잘 사용해보시기 바랍니다.
